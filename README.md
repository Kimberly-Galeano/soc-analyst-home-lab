## Objective
The objective of this lab was to analyze Linux authentication logs to identify
successful and failed login attempts, simulating basic SOC analyst monitoring tasks.

## Tools Used
- Ubuntu Linux
- Bash (Linux command line)
- Native Linux log files

## Lab Steps
1. Accessed authentication logs located at /var/log/auth.log
2. Identified failed login attempts using grep
3. Identified successful login attempts
4. Captured screenshots as evidence
5. Documented findings

## Skills Demonstrated
- Linux command-line usage
- Log analysis
- Security monitoring fundamentals
- Documentation and reporting
